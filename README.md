# GHMN 

## Get Started

1. Install Pytorch and necessary dependencies.
```
pip install -r requirements.txt
```
2. Extract dataset_info.zip within the daily_processed_dataset directory. You can find the data acquisition and cleaning scripts in the process_data folder. Once processed, ensure all data files are moved to the ./dataset directory.

3. Train and evaluate the model with the following scripts.

```
python train_GHMN.py
```
